vt::timetrigger::TimeTriggerManager struct
#include <src/vt/timetrigger/time_trigger_manager.h>

A time-based manager of triggers that each have an associated time period that get triggered from the progress function.

Contents

  • Reference
    • Base classes
    • Public types
    • Constructors, destructors, conversion operators
    • Public functions

Timed triggers are "drifting", meaning that they approximate the associated time period but are not absolutely fixed in time.
