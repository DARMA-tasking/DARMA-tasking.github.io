Send Message Active Function (type-safe handler) module

Send a message to a auto-registered type-safe active message handler. This is the predominant way that the messenger is expected to be used.

Contents

  • Reference
    • Functions
struct MyMsg : vt::Message {
  explicit MyMsg(int in_a) : a_(in_a) { }
  int a_;
};

void myHandler(MyMsg* msg) {
  // do work ...
}

void sendCode() {
  // myHandler is automatically registered with the overload
  theMsg()->sendMsg<MyMsg, myHandler>(1, msg);
}
