vt::epoch::EpochManip struct
#include <src/vt/epoch/epoch_manip.h>

Class used to manipulate the bits in a EpochType and manage the current sequential IDs for allocating epochs.

Contents

  • Reference
    • Public static functions

Used by the system mostly to manage the bits inside an EpochType. It knows how to set the appropriate bits to change the static type of an EpochType by setting the bit pattern.
