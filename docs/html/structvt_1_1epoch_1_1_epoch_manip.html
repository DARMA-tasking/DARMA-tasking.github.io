vt::epoch::EpochManip struct
#include <src/vt/epoch/epoch_manip.h>

Component for managing epoch ID allocation/deallocation and manipulating the bits inside an epoch identifier.

Contents

  • Reference
    • Base classes
    • Public types
    • Public static functions
    • Constructors, destructors, conversion operators
    • Public functions

Used by the system mostly to manage the bits inside an EpochType. It knows how to set the appropriate bits to change the type bits of an EpochType by setting the bit pattern.
