vt::vrt::collection::balance::NodeLBData struct
#include <src/vt/vrt/collection/balance/node_lb_data.h>

A VT component that backs the instrumentation of virtualized entities on each node, such as the objects that the collection manager orchestrates, to provide data to the load balancing framework. The actual instrumentation occurs in vt::vrt:collection::balance::ElementLBData which is composed into the elements of a collection.

Collects LB data/timings on active function/methods for objects and communication between them on each node. After collecting this data, passes it to the load balancing framework, specifically the * vt::vrt::collection::balance::LBManager
