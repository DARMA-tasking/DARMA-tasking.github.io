auto registerComponent(T** ref, typename BaseComponent::DepsPack<Deps...>, Cons && ... cons) -> registry::AutoHandlerType
Idempotent registration of a component into this runtime pack. Component dependencies specified with variadic template parameters in DepsPack. Registration does not imply the component will be created; it must be added subsequently to be enabled. It simply declares its existence and connectivity with other components.

Idempotent registration of a component into this runtime pack. Component dependencies specified with variadic template parameters in DepsPack. Registration does not imply the component will be created; it must be added subsequently to be enabled. It simply declares its existence and connectivity with other components.
