59 /*
60  *
61  * This implementation is directly based on the Dijkstra-Scholten parental
62  * responsibility termination algorithm detailed in this paper based on building
63  * an engagement tree limited to the size of nodes engaged. In computations
64  * where the fanout is low and well-bounded (and thus theoretically scalable),
65  * this algorithm performs optimally wrt the number of nodes that communicate.
66  *
67  * "Termination Detection for Diffusing Computations"
68  * https://www.cs.mcgill.ca/~lli22/575/termination3.pdf
69  *
70  * Algorithmic-specific fault tolerance can be attained for this algorithm based
71  * on this paper that extends the Dijkstra-Scholten algorithm in two ways---one
72  * for independent failures, the other from related process faults:
73  *
74  * "Adoption Protocols for Fanout-Optimal Fault-Tolerant Termination Detection"
75  * http://jlifflander.com/papers/ppopp13termination.pdf
76  */
