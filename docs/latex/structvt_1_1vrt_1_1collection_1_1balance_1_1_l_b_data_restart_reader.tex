\doxysubsection{Detailed Description}
A VT component for reading a user-\/input mapping file for object to node. 

A common flow is for the LBAF (Load Balancing Analysis Framework) to generate a new load distribution offline, producing LB data file, which are then read by this component to follow the LB distribution described in those mapping files.
