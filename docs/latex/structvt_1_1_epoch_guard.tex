\doxysubsection{Detailed Description}
An epoch wrapper that provides an RAII-\/style mechanism for owning an epoch for the duration of a scope block. 

When \mbox{\hyperlink{structvt_1_1_epoch_guard}{Epoch\+Guard}} is created, it pushes the given epoch onto the epoch stack. When \mbox{\hyperlink{structvt_1_1_epoch_guard}{Epoch\+Guard}} is destroyed, it pops the epoch. It can explicitly pop the epoch early.

\mbox{\hyperlink{structvt_1_1_epoch_guard}{Epoch\+Guard}} is non-\/copyable.
