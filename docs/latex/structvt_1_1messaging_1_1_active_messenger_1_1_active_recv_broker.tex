\doxysubsection{Detailed Description}
Active receive broker that keeps a pool of pre-\/posted Irecv slots per size-\/class tag to drain unexpected \mbox{\hyperlink{structvt_1_1messaging_1_1_active_msg}{Active\+Msg}} traffic quickly. 

Each completed slot dispatches directly to finish\+Pending\+Active\+Msg\+Async\+Recv and is immediately reposted with a fresh buffer of the same capacity.
