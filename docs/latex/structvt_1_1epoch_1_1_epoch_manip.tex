\doxysubsection{Detailed Description}
Component for managing epoch ID allocation/deallocation and manipulating the bits inside an epoch identifier. 

Used by the system mostly to manage the bits inside an {\ttfamily \mbox{\hyperlink{structvt_1_1epoch_1_1_epoch_type}{Epoch\+Type}}}. It knows how to set the appropriate bits to change the type bits of an {\ttfamily \mbox{\hyperlink{structvt_1_1epoch_1_1_epoch_type}{Epoch\+Type}}} by setting the bit pattern.
