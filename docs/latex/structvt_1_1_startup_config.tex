Preconfigure VT with argc/argv. This will remove all VT arguments and create a {\ttfamily \mbox{\hyperlink{structvt_1_1_startup_config}{Startup\+Config}}} for VT that should be passed to {\ttfamily initialize\+Preconfigured}. Optionally, one may specify an M\+PI communicator to use (otherwise, it defaults to {\ttfamily M\+P\+I\+\_\+\+C\+O\+M\+M\+\_\+\+W\+O\+R\+LD}). 

\begin{DoxyNote}{Note}
M\+PI must be initialized to call this function because if an error occurs it uses M\+PI rank to limit how many times the error text gets printed.
\end{DoxyNote}
